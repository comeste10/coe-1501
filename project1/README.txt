INTRODUCTION

This project implements a HugeInteger class which mimics some functionality of
JAVA's BigInteger class.
 
INSTALLATION

The following files should be present:
  
  TestHugeInteger.java
  HugeInteger.java  

Compile with:

  javac TestHugeInteger.java

USAGE

The program should compile and run with Java 2 SDK 1.5 or later:

  java TestHugeInteger

KNOWN PROBLEMS

None.
