INTRODUCTION

This project implements a dynamic programming algorithm to find an optimal cut sequence
on a bolt of cloth, given a set of patterns.
 
INSTALLATION

The following files should be present:
  
  TestClothCutter.java
  ClothCutter.java
  Cloth.java
  Cut.java
  Garment.java
  Pattern.java
  Rectangle.java  

Compile with:

  javac TestClothCutter.java

USAGE

The program should compile and run with Java 2 SDK 1.5 or later:

  java TestClothCutter

KNOWN PROBLEMS

None.
