INTRODUCTION

This project implements a dynamic programming algorithm to find an optimal
binary search tree, given a set of keys and corresponding frequencies. The
optimal BST is determined by comparison of weighted internal path lengths.
 
INSTALLATION

The following files should be present:
  
  TestBSTOptimizer.java
  BSTOptimizer.java
  BinaryTree.java
  Node.java

Compile with:

  javac TestBSTOptimizer.java

USAGE

The program should compile and run with Java 2 SDK 1.5 or later:

  java TestBSTOptimizer

KNOWN PROBLEMS

None.
